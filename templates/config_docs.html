Configuration Manual

Introduction Structure Available Steps Variables Examples

Introduction

This manual describes how to create configuration files for new webshops. Each webshop has its own JSON configuration file that defines how prices should be calculated.

Structure

A configuration file has the following basic structure:

{
    "domain": "example.com",
    "categories": {
        "square_meter_price": {
            "steps": [
                // Steps go here
            ]
        },
        "shipping": {
            "steps": [
                // Steps go here
            ]
        }
    }
}

Available Steps

select

Selects a value from a dropdown or radio button list.

Parameter Type Description
selector string CSS selector for the element
value string Value to select (can contain variables)
unit string Unit of the value (mm/cm)
option_container string CSS selector for custom dropdown container
option_selector string CSS selector for custom dropdown options 
{
    "type": "select",
    "selector": "#thickness",
    "value": "{thickness}",
    "unit": "mm"
}

input

Fills a value in an input field.

Parameter Type Description
selector string CSS selector for the input field
value string Value to fill in (can contain variables)
unit string Unit of the value (mm/cm/m)
clear_first boolean Whether to clear the field before typing (default: true) 
{
    "type": "input",
    "selector": "#width",
    "value": "{width}",
    "unit": "mm"
}

click

Clicks on an element.

Parameter Type Description
selector string CSS selector for the element to click
description string Optional description of what this click does 
{
    "type": "click",
    "selector": "#calculate-button"
}

wait

Waits for a specified time before executing the next step.

Parameter Type Description
duration string/number Wait time in seconds or predefined value
Predefined wait times:
  • short: 0.5 seconds
  • default: 1.0 second
  • long: 1.5 seconds
  • longer: 3.0 seconds
{
    "type": "wait",
    "duration": "long"
}

blur

Blurs (unfocuses) an input field, which can trigger validation or calculations on some websites.

Parameter Type Description
selector string CSS selector for the element to blur 
{
    "type": "blur",
    "selector": "#width"
}

modify_element

Modifies an element's attributes or executes custom JavaScript on it. Useful for fixing hard-to-click elements or working around website limitations.

Parameter Type Description
selector string CSS selector for the element to modify
add_class string CSS class to add to the element
add_attribute object Key-value pairs of attributes to add
script string Custom JavaScript to run on the element 
{
    "type": "modify_element",
    "selector": ".cart",
    "script": "element.style.zIndex = '9999'; element.style.position = 'relative';"
}

Tip: The modify_element step is particularly useful for dealing with elements that are difficult to click because they're obscured or have other interaction issues. The element is accessible in the script via the variable element.

read_price

Reads a price from an element on the page.

Parameter Type Description
selector string CSS selector for the element containing the price
includes_vat boolean Whether the price includes VAT (true) or not (false)

Note: If the price element is not found, the system will return 0.00 as the price instead of failing. This is particularly useful for handling optional costs like shipping.

{
    "type": "read_price",
    "selector": ".price-total",
    "includes_vat": true
}

captcha

Handles captchas including Google reCAPTCHA and standard checkbox captchas. Useful for forms that include captcha verification.

Parameter Type Description
captcha_type string Type of captcha: 'checkbox' (default) or 'recaptcha_v2'
selector string CSS selector for the captcha checkbox (leave empty for auto-detection)
frame_selector string CSS selector for the iframe containing the captcha
skip_on_failure boolean Whether to continue with next steps if captcha solving fails (default: true) 
{
    "type": "captcha",
    "captcha_type": "checkbox",
    "frame_selector": "iframe[src*='recaptcha']"
}

Note: For standard "I'm not a robot" checkbox captchas, you can often leave the selector field empty to use auto-detection. If the captcha is in an iframe (common with Google reCAPTCHA), providing the frame_selector will help locate it more reliably.

Variables

You can use the following variables in the configuration:

  • {thickness}: Thickness in mm
  • {width}: Width in mm
  • {length}: Length in mm
  • {quantity}: Number of pieces

Note: The unit of the variables can be adjusted using the unit parameter in the step configuration.

Examples

Example 1: Basic configuration

{
    "domain": "example.com",
    "categories": {
        "square_meter_price": {
            "steps": [
                {
                    "type": "select",
                    "selector": "#thickness",
                    "value": "{thickness}",
                    "unit": "mm"
                },
                {
                    "type": "wait",
                    "duration": "default"
                },
                {
                    "type": "input",
                    "selector": "#width",
                    "value": "{width}",
                    "unit": "mm"
                },
                {
                    "type": "wait",
                    "duration": "default"
                },
                {
                    "type": "input",
                    "selector": "#length",
                    "value": "{length}",
                    "unit": "mm"
                },
                {
                    "type": "wait",
                    "duration": "default"
                },
                {
                    "type": "read_price",
                    "selector": ".price",
                    "includes_vat": false
                }
            ]
        }
    }
}

Example 2: Complex configuration with click issues

{
    "domain": "example.com",
    "categories": {
        "square_meter_price": {
            "steps": [
                {
                    "type": "select",
                    "selector": ".thickness-select",
                    "value": "{thickness}",
                    "unit": "mm"
                },
                {
                    "type": "wait",
                    "duration": "long"
                },
                {
                    "type": "input",
                    "selector": "#width",
                    "value": "{width}",
                    "unit": "cm"
                },
                {
                    "type": "blur",
                    "selector": "#width"
                },
                {
                    "type": "wait",
                    "duration": "short"
                },
                {
                    "type": "input",
                    "selector": "#length",
                    "value": "{length}",
                    "unit": "cm"
                },
                {
                    "type": "wait",
                    "duration": "default"
                },
                {
                    "type": "modify_element",
                    "selector": ".cart-button",
                    "script": "element.style.zIndex = '999'; element.style.position = 'relative';"
                },
                {
                    "type": "click",
                    "selector": ".cart-button"
                },
                {
                    "type": "wait",
                    "duration": "longer"
                },
                {
                    "type": "read_price",
                    "selector": "#total-price",
                    "includes_vat": true
                }
            ]
        },
        "shipping": {
            "steps": [
                {
                    "type": "input",
                    "selector": "#postcode",
                    "value": "1000AA"
                },
                {
                    "type": "click",
                    "selector": "#calculate-shipping"
                },
                {
                    "type": "wait",
                    "duration": "long"
                },
                {
                    "type": "read_price",
                    "selector": ".shipping-cost",
                    "includes_vat": true
                }
            ]
        }
    }
}
